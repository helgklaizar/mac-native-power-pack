# 🎬 Video Alchemist V1 (Native Video Build)

This assembly is built for **local video generation and temporal processing** using Metal.

## 🧩 Из чего собирается (Components)
1.  **[svd-mlx](https://github.com/lucasnewman/svd-mlx)**: Нативный порт Stable Video Diffusion. Позволяет генерировать короткие видеоролики из изображений прямо на Mac.
2.  **[mlx-video](https://github.com/ml-explore/mlx-video)**: Набор инструментов для покадровой обработки и применения диффузионных эффектов к видеопотоку.
3.  **Metal Upscaler**: Нативные фильтры для повышения разрешения кадров в реальном времени.

## 🏆 Где применять:
- Креативный видеомонтаж с ИИ-эффектами.
- Локальная генерация анимаций для соцсетей.
- Реставрация старых видеороликов.
