# 🐜 Tiny Titan V1 (Edge & Mobile Build)

This assembly is focused on **ultra-portability** and running AI on the most constrained hardware (MacBook Air 8GB, iPad, IoT).

## 🧩 Из чего собирается (Components)
1.  **[Llama-3.2-1B / 3B]**: Оптимизированные веса сверхмалых моделей.
2.  **Ultra-Quant (2-bit)**: Экстремальное сжатие весов для работы в 1-2 ГБ оперативной памяти.
3.  **Low-Power Metal Core**: Профили энергопотребления для максимального времени жизни от батареи.

## 🏆 Где применять:
- Встраиваемые системы на базе Apple Silicon.
- Оффлайн-ассистенты для путешествий.
- Фоновые процессы мониторинга с минимальным расходом заряда.
