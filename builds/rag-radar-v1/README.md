# 🛸 RAG Radar V1 (Search & Retrieval Build)

This assembly is optimized for **Semantic Search, Embeddings, and local Knowledge Retrieval**.

## 🧩 Из чего собирается (Components)
1.  **[mlx-embeddings]**: Использование моделей типа BGE или Stella для генерации высококачественных векторов на GPU.
2.  **LanceDB-Local**: Высокопроизводительная векторная база данных, работающая напрямую с файловой системой Mac.
3.  **Ranker-Native**: Нативные Cross-Encoder ядра для переранжирования результатов поиска.

## 🏆 Где применять:
- Локальные базы знаний (Second Brain).
- Поиск по гигантским кодовым базам (100k+ файлов).
- Приватные поисковики по личным документам (PDF, Markdown).
